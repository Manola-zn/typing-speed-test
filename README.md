# ⌨️ Typing Speed Test App

A modern typing speed tester built with React and TypeScript. It helps users measure their typing speed (WPM), track time and improve accuracy with real-time feedback and optional sound effects.

## ✨ Features
- Live WPM calculation
- Timer tracking
- Key sound effects
- Dark/Light mode toggle
- Settings modal
- Responsive glassmorphism UI

## 🛠️ Built With
- React
- TypeScript
- CSS3
- Bootstrap Icons
